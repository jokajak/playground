WCA Scorecard Generator

Print official-style WCA scorecards for a fake competition. Type the competition, event, round, group, name, ID, time limit and cutoff directly into each card, or leave any of them blank to fill in by hand — results and signatures are always hand-written.

Prefill from a list of names

One name per line becomes one card, in that order — this replaces whatever's in Scorecards above with the number of lines. Add a WCA ID after a comma to use it as-is; otherwise one is generated in the standard format (YEAR + first four letters of the last name, padded with X if shorter, + a two-digit sequence number that increments for repeated year+name combinations). A single-word name is treated as its own last name.

Each card's info fields are independently editable, so different cards can have different names, IDs, or even competitions. Formats set the attempt count (Bo1 = 1, Bo2 = 2, Bo3/Mo3 = 3, Ao5 = 5). The cutoff variant is still 5 rows, with a marked line after attempt 2 — a competitor must beat the cutoff in one of the first 2 attempts to earn attempts 3–5. Extra rows are blank spares for replacement attempts. Results and signatures are always blank to hand-write. Cards print four to a page.
