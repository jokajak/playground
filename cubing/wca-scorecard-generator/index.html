WCA Scorecard Generator

Print blank, official-style WCA scorecards for a fake competition — then fill in the competition, event, names and results by hand.

Every text field is a blank line to write on. Formats set the attempt count (Bo1 = 1, Bo2 = 2, Bo3/Mo3 = 3, Ao5 = 5). The cutoff variant is still 5 rows, with a marked line after attempt 2 — a competitor must beat the cutoff in one of the first 2 attempts to earn attempts 3–5. Extra rows are blank spares for replacement attempts. Cards print four to a page.
