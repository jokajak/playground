# Megaminx Last Layer — J Perm Beginner Method

A printable, one-page reference for the megaminx last-layer steps from J Perm's
*How to Solve a Megaminx! [Beginner Tutorial]* — the Gray Star (edge orientation)
and Align Star Pieces (edge permutation) algorithms shown in the video, plus the
two corner steps (Gray Face and Solve the Megaminx) filled in with the standard
beginner algorithms. A second page collects a few alternate/extra cases for the
corner steps.

Each case has a schematic gray-face diagram (edges as circles, corners as
pentagons; gray = oriented, dark = not yet, light = don't-care) and highlights the
sexy / Sune triggers the algorithms are built from.

## Use it

Open `index.html` directly in your browser, then use the **Print** button
(top right) — the page is laid out for Letter-size sheets, with the extra
algorithms on a second page.
