# 2-Look CFOP Cheat Sheet

A printable, one-page reference for the 2-look OLL/PLL last-layer algorithms
(algorithms from J Perm; alternates simulator-verified), plus a finger-trick
trigger table (sexy move, sledgehammer, Sune trigger, etc.) that most of the
algorithms are built from.

## Use it

Open `index.html` directly in your browser, then use the **Print** button
(top right) — the page is laid out for a single Letter-size sheet.
