Playground

Small, self-contained web utilities. No build step, no accounts.

Games & Tools

  • Jeopardy Maker

    A Jeopardy clone for 2–6 players on a shared screen, with a built-in board editor.

  • Bracket Maker

    Generates printable single-elimination tournament brackets (2–64 participants).

  • QR Code Maker

    Generates QR codes, with optional embedding onto an uploaded picture.

  • OpenGrid Planner

    Plans OpenGrid wall layouts and tile counts, optimized for BambuLab printer bed sizes.
Cubing

  • 1×2×3 Character Cube Scrambler

    Uniform random-state scrambles for the 1×2×3 puzzle, rendered as a character cube.

  • 2-Look CFOP Cheat Sheet

    Printable OLL/PLL algorithm reference with a finger-trick trigger table.

  • WCA Scorecard Generator

    Prints blank, official-style WCA competition scorecards to fill in by hand.
