QR Code Maker

Generate a QR code, optionally merged into a picture. Everything runs in your browser — nothing is uploaded.

Embed in a picture (optional)

Pick a photo or graphic and the QR code is merged into it. Drag the code on the preview to reposition it.

Merge reshapes the picture's own pixels to carry the code — nothing is pasted on top, only the corner marks every scanner needs. Center logo makes a clean code, tinted to the image's main color, with the image in the middle — best for logos. Stamp is a solid tile: least pretty, most scannable.

Error correction was raised to H — recommended when the code sits on a busy image.
